## How to run
1) Create a virtual environment (optional but recommended)
   python -m venv .venv
   # Windows: .venv\Scripts\activate
   # macOS/Linux: source .venv/bin/activate

2) Install Flask
   pip install flask

3) Run the server
   python app.py

4) Open your browser at
   http://127.0.0.1:5000

## Customize
- Edit text and sections in templates/*.html
- Change colors or styles in static/style.css
- Replace images in static/images/
